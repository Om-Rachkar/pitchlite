PITCHLITE
-A One-Minute Pitch Builder
:TEAM HACKOHOLICS
---

PROBLEM STATEMENT:
Design and develop a web application that guides users in creating a structured one-minute 
pitch using prompts and generates a clean, presentable summary.

---

INTRO:
Pitchlite is a simple and powerful web tool that helps users craft concise, structured,
and impactful one-minute pitches. Whether for startups, interviews, or presentations,
Pitchlite guides you through the process of turning ideas into clear summaries.

---

 FEATURES:
- Prompt-driven input: Enter your idea and get guided through Problem → Solution → Value → Call to Action.
- Live Preview: See your pitch update in real time.
- Concise Output: Generates a polished one-minute pitch summary.
- Export Options: Copy, save.
- Responsive Design: Works seamlessly on desktop.
- Pitches in 60 seconds.
- Q&A Related to Pitch.
  
---

 WORKFLOW:

-User input:
User enters key details:Problem,solution,audience,traction and call to action through an intuitive form interface.

-Processing:
Javascript logic applies predefined pitch formulas and templates to structure content based on selected style and sector.

-Generation:
System generates a polished one-minute pitch with real time preview updates as users make changes.

-Output:
Live preview with speech playback,copy to clipboard, and save options for immediate use.
